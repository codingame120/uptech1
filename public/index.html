Removing Termux Welcome Screen Text We will start with removing termux welcome screen text. Type: 1 cd .. To change directory. 1 ls To list directory. 1 cd usr/etc Change directory to etc 1 ls List directory. 1 rm -rf motd Remove file named motd. When you start new window you will see that there is no more welcome text and termux start with clean window.
